Agent coding life hack:

Use this handy script I made called "mirror_cc_skills" to make your life (very slightly) easier:

https://github.com/Dicklesworthstone/misc_coding_agent_tips_and_scripts?tab=readme-ov-file#mirror-claude-code-skills

Install with the curl | bash one-liner. 

Once installed, you can simply type "mirror_cc_skills" in any project folder that has local Claude Skills defined at these will be mirrored in your global folder so that they're available in all your sessions in any project.
