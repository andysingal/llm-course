[wikiwise](https://github.com/TristanH/wikiwise)

A native macOS app that turns any folder of markdown files into a browsable, publishable wiki — maintained by your coding agent.

[apfel](https://github.com/Arthur-Ficial/apfel)

Apple Silicon Macs ship a built-in LLM via Apple FoundationModels. apfel exposes it as a UNIX tool and a local OpenAI-compatible server. 100% on-device. No API keys, no cloud.
